## ğŸ—ï¸ Development Process: "The 24hr Vibe"

The project was built using an "Agentic Pair Programming" approach, mirroring Innovatr's own philosophy of rapid iteration:

1.  **Foundation:** Initializing a lightning-fast Vite + React 19 environment.
2.  **Design System:** Establishing a custom Tailwind configuration for brand-specific colors (`brand-blue`, `brand-orange`) and claymorphism utilities.
3.  **Kinetic Layer:** Integrating GSAP early to ground the experience in characterful motion rather than static pixels.
4.  **Refinement:** Unifying sections (Hero & Features) to eliminate friction and focus on the core USP: _Speed_.
5.  **Dark Mode:** Implementing a robust theme-switching architecture to double the visual utility of the design.
